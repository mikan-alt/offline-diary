# Offline Diary - iPhone運用メモ

## 何が入っているか

- `index.html`：日記アプリ本体
- `sw.js`：オフライン起動用キャッシュ
- `manifest.webmanifest`：ホーム画面アプリ化用
- `icon-192.png`, `icon-512.png`：アイコン

## 重要

日記本文はサーバーに保存されません。ブラウザ内の IndexedDB に保存されます。
そのため、定期的にアプリ内の「データ」→「JSONを書き出す」でバックアップしてください。

## 推奨運用

1. GitHub Pages にこのフォルダの中身を置く
2. iPhoneのSafariで公開URLを開く
3. 共有ボタン → 「ホーム画面に追加」
4. 一度起動して、日記を書いて保存
5. 以後はホーム画面アイコンからオフラインで使う

## バックアップ

- アプリ下部の「バックアップ」または「データ」→「JSONを書き出す」
- 保存先は「このiPhone内」や外部ストレージ推奨
- 復元は「データ」→「JSONを読み込む」

## 注意

URLやドメインを変えると、別アプリ扱いになり、以前のIndexedDBデータが見えなくなります。
移転前に必ずJSONを書き出してください。
